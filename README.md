# microframework

 **Server Requirements** <br/>
 - PHP >= 7.2.0
 - PHP PDO Extension
 - PHP XML Extension
 - PHP Mysql extension
 
 **Install Dependencys** <br />
 composer install
 
 **Script SQL Database** <br />
  app/config/banco.sql
  
 **Local Development Server** <br/>
 php -S localhost:9999
